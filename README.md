# Minaya Algora — Portfolio

Personal portfolio site. Static HTML/CSS/JS, no build step.

## Local development

Open `index.html` in a browser. That's it.

## Structure

```
portfolio/
├── index.html       # Home
├── about.html       # Bio + manifesto + tools + personal
├── work.html        # Case studies + timeline
├── contact.html     # Contact + CV download
├── styles.css       # Design system
├── script.js        # Scroll reveals + nav
└── assets/
    ├── images/      # Photos
    └── docs/        # CV PDF
```

## Deploy

Connected to Vercel/Netlify — every push to `main` auto-deploys.

## Stack

Plain HTML, CSS, and vanilla JS. Google Fonts (Fraunces + Inter). No frameworks, no dependencies.
